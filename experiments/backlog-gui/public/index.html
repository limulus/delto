API

  • /api/snapshot — backlog + journal + stats + intents
  • /api/backlog — enriched items, eligibility, dependents
  • /api/journal — completed entries + velocity stats
  • /api/journal/:file — one entry, rendered to HTML
  • /api/intents — the intent queue (GET) / record intent (POST)
